# \# EduStream - E-Learning Management System

# 

# \## Description

# A comprehensive web-based E-Learning Management System developed to streamline the learning process. Built with a focus on user-friendly navigation and efficient data management.

# 

# \## Features

# \*   \*\*User Authentication:\*\* Secure Signup and Login functionality.

# \*   \*\*Course Management:\*\* Browse and access various learning modules.

# \*   \*\*Database Integration:\*\* Powered by MySQL for reliable data persistence.

# 

# \## Technologies Used

# \*   \*\*Frontend:\*\* HTML, CSS

# \*   \*\*Backend:\*\* PHP

# \*   \*\*Database:\*\* MySQL

# 

# \## How to Run

# 1\. Clone this repository to your local machine.

# 2\. Ensure you have a local server environment (like XAMPP or WAMP) running.

# 3\. Import the provided `.sql` file into your MySQL database.

# 4\. Place the project files in your `htdocs` (or equivalent) folder.

# 5\. Configure your database connection in `db.php`.

# 6\. Access the system via your web browser.
